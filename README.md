# 🎓 Student Grade Calculator

This program calculates a student's grade based on marks and bonus marks.

## ✨ Features

* ➕ Adds bonus marks to the student's score
* 🏆 Assigns grades based on total marks
* ✅ Determines Pass/Fail status
* 💬 Provides performance feedback using a `switch` statement

## 📊 Grade Criteria

| Marks Range  | Grade      |
| ------------ | ---------- |
| 90 and Above | 🥇 Grade A |
| 70 - 89      | 🥈 Grade B |
| 50 - 69      | 🥉 Grade C |
| Below 50     | ❌ Grade F  |
